# Machine_Learning_projects
A collection of machine learning projects including food delivery time prediction, global pollution analysis, and energy recovery — showcasing different algorithms and performance comparisons.


A curated collection of applied ML projects exploring predictive modeling, data analysis, and algorithm comparisons.  

## Projects Included
- **Food Delivery Time Predictor** – Estimate delivery times using Linear & Logistic Regression.  
- **Global Pollution Analysis** – Analyze environmental datasets to uncover trends and insights.  
- **Energy Recovery Models** – Apply ML algorithms to optimize energy recovery and compare performance.  

## Highlights
- Data preprocessing & feature engineering  
- Implementation of multiple ML algorithms  
- Model evaluation & performance comparison  
- Well-documented and reproducible code
